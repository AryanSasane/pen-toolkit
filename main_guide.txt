def get_session function
# most imp fn, DVWA reqs login, this fn logs in & rets an authenticated session 
# session = reques... creates a new session object, kinda like opening a browser, all reqs made through this sesh share the same cookies
# response = session... first we GET the login page before posting creds, necessary coz DVWA uses CSRF token, a hidden sec field that must be included with the login form
# if "user_token" in response.text... checks if the page has a CSRF token, DVWA includes a hidden field called "user_token" to prevent automated logins, we need to extract it & include it in our POST reqs
# match = re.search... uses regex to find & extract the token val from the HTML, the pattern looks for user_token' value='...' & captures whats in the quotes
# token = match.group(1) extracts the captured group - the actual group value
# session.post... posts the login form with the creds and the CSRF token, after the sesh is auth, all subsequents reqs will be logged in
